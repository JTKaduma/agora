## Design Sections
1. Profile Picture
   - Display current profile image in a circular avatar
   - Edit button/icon overlay for upload/change image
   - Tooltip text: "Update your profile photo"

2. Personal Information
   - Name field (editable)
   - Email field (editable if allowed, or read-only with notice)
   - Clear label/placeholder strategy
   - Validation right away (e.g. email format)

3. Password / Security
   - Change password controls (current password, new password, confirm password)
   - Password strength indicator
   - Two-factor auth prompt (mobile/email)

4. Notifications
   - Basic notification preferences:
     - Email notifications toggle
     - App/push notifications toggle
     - Security alerts toggle
   - Save changes primary action with persistent state

## Compliance Notes
- Followed established style in Figma components and spacing system.
- Kept typography and color usage consistent with design system.
- Prioritized accessibility: labels, focus, sufficient contrast.
